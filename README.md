# Traffic Accident Risk Analysis 🚦

## 📌 Project Overview
This project analyzes urban traffic accident risk using Databricks and PySpark.

## 🛠 Tools & Technologies
- Databricks
- PySpark
- SQL

## 📊 Key Features
- Built medallion architecture (Bronze, Silver, Gold)
- Data cleaning and transformation
- Analysis of accident hotspots and risk factors

## 📈 Insights
- Identified high-risk locations and road types
- Impact of weather on accident severity
- Driver demographics and violation patterns
